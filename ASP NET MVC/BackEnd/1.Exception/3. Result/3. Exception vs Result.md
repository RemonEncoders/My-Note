# ৪. Exception Flow vs Result Flow

---

# Exception Flow

```text
	Service
	  |
	  |(THROW)
	  ↓ 
	Controller catch
	  |
	  ↓
	JSON
```

---

# Result Flow

```text
	Service
	  ↓ (RETURN Result.Fail)
	Controller checks IsSuccess
	  ↓
	JSON
```
