# ২. Result Pattern এ কী হয়?

এখন failure exception হিসেবে না গিয়ে DATA হিসেবে return হয়।

---

# Service

```csharp
public Result<User> GetUser(int id)
{
    var user = _db.Users.Find(id);

    if(user == null)
    {
        return Result.Fail<User>("User Not Found");
    }

    return Result.Ok(user);
}
```

---

# Controller

```csharp

	var result = _service.GetUser(id);
	
	if(!result.IsSuccess)
	{
	    return Json(new
	    {
	        success = false,
	        message = result.Error
	    });
	}
	
	return Json(new
		{
			success = true,
			data = result.Value
		});
```

---

# ৩. Important Insight

আগে:

```text
Failure communicated by THROWING
```

এখন:

```text
Failure communicated by RETURNING
```
