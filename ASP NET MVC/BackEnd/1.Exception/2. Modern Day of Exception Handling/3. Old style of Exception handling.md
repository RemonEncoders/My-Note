# ১৬. Important Truth About try-catch Performance

অনেকে ভুল ভাবে:

> try-catch থাকলেই slow

না।

Exception THROW হওয়া expensive।

try block expensive না।

---

# Exception expensive কেন?

কারণ:

- stack unwinding
- stack trace generation
- memory allocation

তাই exception দিয়ে business flow control করা bad practice।

# ১৭. Production Grade Exception Strategy

Senior systems এ normally থাকে:

---

## 1. Global Exception Middleware

Unhandled exception catch।

---

## 2. Custom Exception Types

```cs
public class BusinessException : Exception
{

}
```

---

## 3. Exception Mapping

```cs
ValidationException → 400
NotFoundException → 404
UnauthorizedException → 401
```

---

## 4. Structured Logging

```cs
_logger.LogError(ex,
    "Order creation failed for user {UserId}",
    userId);
```

NOT:

```cs
_logger.LogError(ex.Message);
```
