# Analog Signal vs Digital Signal

| **Feature**      | **Analog Signal**                   | **Digital Signal**                                |     |
| ---------------- | ----------------------------------- | ------------------------------------------------- | --- |
| **Nature**       | Continuous                          | Discrete (step by step)                           |     |
| **Values**       | Infinite values.                    | Certain Set of Values (Limited values)            |     |
| **Shape**        | Smooth wave                         | Square/rectangular wave                           |     |
| **Data form**    | Real-world natural form<br>(-∞, +∞) | most of the time it is Binary form (0 and 1).<br> |     |
| **Noise effect** | Easily affected by noise            | More resistant to noise                           |     |
| **Accuracy**     | Lower after long distance           | Higher and more reliable                          |     |
| **Processing**   | Harder to process                   | Easy to process with computers                    |     |
| **Storage**      | Difficult                           | Easy                                              |     |
| **Example**      | Human voice, FM radio               | Computer data, USB, WiFi                          |     |
