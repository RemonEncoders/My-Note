# ⭐ 2. Based on Channel Capacity Theory

## 🔹 (A) Nyquist Channel (Noiseless condition assumption)

This is used for **noiseless channel capacity**.

### Nyquist Theorem (Important Formula):

$$C = 2B \log_2(L)$$

Where:

- `C` = channel capacity (bps)
- `B` = bandwidth (Hz)
- `L` = number of signal levels
	

---

## Relationship:

$$L = 2^n$$
- **n** = Number of bits per symbol
- `L` = number of signal levels
---

## Therefore:

Each symbol can carry **n bits of information**.

---

### Meaning:

✔ Higher bandwidth → more data rate  
✔ More signal levels → more data per symbol

---

### Use case:

- Ideal (noiseless) digital transmission model

---

## 🔹 (B) Shannon Channel (Noisy channel capacity)

Used when noise is present.

### Shannon Capacity Formula:

$$C = B \log_2(1 + SNR)$$

Where:

- SNR = Signal-to-Noise Ratio

---

### Meaning:

✔ Noise limits maximum data rate  
✔ More SNR → better performance
