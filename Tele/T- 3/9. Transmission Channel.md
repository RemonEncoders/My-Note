# ⭐ 1. Types of Transmission Channels (Core Classification)

## 🔹 (A) Noiseless Channel (Ideal Channel)

✔ No noise present  
✔ No distortion  
✔ Perfect communication (theoretical only)

### Key idea:

> Whatever you send = exactly received

### Example (theory only):

- Perfect fiber link (ideal assumption)
- Mathematical model

---

## 🔹 (B) Noisy Channel (Real Channel)

✔ Noise is always present  
✔ Signal gets distorted  
✔ Errors may occur

### Types of noise:

- Thermal noise
- Crosstalk
- Interference
- Attenuation noise

### Example:

- Mobile network 📱
- WiFi 📶
- Radio communication 📻
