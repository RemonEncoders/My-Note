# 5. ORDER BY 

```sql
ORDER BY SortOrder
```

👉 এটা condition না

👉 এটা মানে:

> “প্রতিটি row-এর value নিয়ে sort করো”

উদাহরণ:

|SortOrder|
|---|
|5|
|1|
|3|

Result:

```sql
1 → 3 → 5
```

---

# 6. Boolean দিয়ে ORDER BY

```sql
ORDER BY SortOrder IS NULL
```

SQL এটাকে convert করে:

|Condition|Value|
|---|---|
|FALSE|0|
|TRUE|1|

👉 তাই:

- 0 (non-null) আগে
- 1 (null) পরে

---

# 7. Multiple ORDER BY (ThenBy logic)

```sql
ORDER BY SortOrder, ProductName
```

👉 মানে:

- প্রথমে SortOrder
- যদি same হয় → ProductName

---

# 8. CASE ব্যবহার (ORDER BY এর power)

```sql
ORDER BYCASE WHEN SortOrder IS NULL THEN 1 ELSE 0 END,SortOrder
```

👉 এখানে CASE তৈরি করছে একটা ranking value

---

# 9. Multiple CASE (একাধিক rule)

```sql
ORDER BYCASE WHEN IsDeleted = 1 THEN 1 ELSE 0 END,CASE WHEN IsActive = 0 THEN 1 ELSE 0 END,SortOrder
```

👉 মানে:

1. Deleted না এমন আগে
2. Active আগে
3. তারপর SortOrder

---

# 10. Nested CASE (decision tree)

```sql
ORDER BY 
	CASE WHEN IsFeatured = 1 
		THEN        
			CASE  WHEN StockQty > 0 
				THEN 
					1            
				ELSE 
					2        
			END    
		ELSE 
			3
	END
```

👉 মানে:

- Featured + stock আছে → first priority
- Featured + stock নেই → second
- Featured না → last

---

# 11. Multiple CASE vs Nested CASE

|Type|মানে|
|---|---|
|Multiple CASE|আলাদা আলাদা priority rule|
|Nested CASE|এক rule এর ভিতরে decision logic|

# 12. SQL কীভাবে sort করে (simple ধারণা)

SQL প্রতিটি row এর জন্য একটা “sort key” বানায়:

```sql
(row) → (key1, key2, key3)
```

তারপর compare করে:

- আগে key1
- same হলে key2
- same হলে key3

👉 এটাকে বলে **lexicographical sorting**

# 13. LINQ mapping

### সাধারণ sorting:

```cs
.OrderBy(x => x.SortOrder)
```

---

### null handle:

```cs
.OrderBy(x => x.SortOrder == null).ThenBy(x => x.SortOrder)
```

---

### SQL equivalent:

```cs
ORDER BYCASE WHEN SortOrder IS NULL THEN 1 ELSE 0 END,SortOrder
```

# 14. বাস্তব জীবনের ব্যবহার

## ERP / Business system:

- pending → approved → rejected
- stock priority
- manual sorting

## E-commerce:

- featured products first
- in-stock first
- best sellers
- তারপর name

## Banking:

- pending transaction first
- overdue next
- completed last
