`ON` এবং `USING` — দুটোই SQL `JOIN`-এ টেবিলগুলোর মধ্যে matching condition নির্ধারণ করার জন্য ব্যবহার করা হয়। তবে এদের মধ্যে কিছু গুরুত্বপূর্ণ পার্থক্য আছে।

## 1. ON Clause

`ON` হলো বেশি flexible এবং সবচেয়ে বেশি ব্যবহৃত।

```sql
SELECT *
FROM Employees e
JOIN Departments d
    ON e.DepartmentId = d.Id;
```

এখানে:

- `Employees.DepartmentId` এবং `Departments.Id` compare করা হচ্ছে।
    
- Column name একই হওয়ার দরকার নেই।
    
- Complex conditionও লেখা যায়।
    

```sql
SELECT *
FROM Orders o
JOIN Customers c
    ON o.CustomerId = c.Id
   AND c.IsActive = 1;
```

---

## 2. USING Clause

`USING` তখন ব্যবহার করা যায় যখন দুই টেবিলে join করার column-এর নাম একই।

ধরো:

### Employees

|Id|DepartmentId|
|---|---|
|1|10|

### Departments

|DepartmentId|Name|
|---|---|
|10|HR|

তখন:

```sql
SELECT *
FROM Employees
JOIN Departments
USING (DepartmentId);
```

এটা internally অনেকটা এর মতো:

```sql
SELECT *
FROM Employees
JOIN Departments
ON Employees.DepartmentId = Departments.DepartmentId;
```

---

## Output এর পার্থক্য

### ON ব্যবহার করলে

```sql
SELECT *
FROM Employees e
JOIN Departments d
ON e.DepartmentId = d.DepartmentId;
```

Result:

|Id|DepartmentId|DepartmentId|Name|
|---|---|---|---|
|1|10|10|HR|

দুই টেবিলের `DepartmentId` আলাদা column হিসেবে থাকবে।

---

### USING ব্যবহার করলে

```sql
SELECT *
FROM Employees
JOIN Departments
USING (DepartmentId);
```

Result:

|Id|DepartmentId|Name|
|---|---|---|
|1|10|HR|

Duplicate join column একবারই দেখাবে।

---

## Multiple Columns

`USING` একাধিক column-ও support করে:

```sql
SELECT *
FROM TableA
JOIN TableB
USING (CompanyId, BranchId);
```

Equivalent:

```sql
SELECT *
FROM TableA
JOIN TableB
ON TableA.CompanyId = TableB.CompanyId
AND TableA.BranchId = TableB.BranchId;
```

## SQL Server সম্পর্কে গুরুত্বপূর্ণ তথ্য

যদি তুমি **SQL Server (MSSQL)** ব্যবহার করো (যা ASP.NET Core + Entity Framework-এ সাধারণত হয়), তাহলে:

```sql
USING (...)
```

**support করে না।**

তোমাকে সবসময় `ON` ব্যবহার করতে হবে:

```sql
SELECT *
FROM Employees e
JOIN Departments d
ON e.DepartmentId = d.DepartmentId;
```

`USING` বেশি দেখা যায়:

- MySQL
    
- PostgreSQL
    
- Oracle
    
- SQLite
    

---

## সহজে মনে রাখার উপায়

|ON|USING|
|---|---|
|SQL Standard, সর্বত্র support|সব DB support করে না|
|Column name আলাদা হতে পারে|Column name একই হতে হবে|
|Complex condition লেখা যায়|শুধু equality join|
|Duplicate column দুবার আসতে পারে|Join column একবার আসে|
|SQL Server-এ ব্যবহার করা যায়|SQL Server-এ ব্যবহার করা যায় না|

তুমি যদি ASP.NET Core + Entity Framework + SQL Server নিয়ে কাজ করো, তাহলে বাস্তবে প্রায় সব JOIN-এ `ON`-ই ব্যবহার করবে।
