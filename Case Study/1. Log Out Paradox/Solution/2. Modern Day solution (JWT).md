
# Modern Solution → Stateless Authentication

এখানে আসে:  JWT (JSON Web Token)-এর মতো স্টেটলেস মেকানিজম ডিস্ট্রিবিউটেড সিস্টেমে  কাজ করে 

---

# JWT কী করে?

Login successful হলে:

```text
Server creates signed token
```

Example:

```text
{
  "userId": 10,
  "role": "Admin",
  "exp": 123456
}
```

এটা cryptographically sign করা হয়।

তারপর token browser/client এ পাঠানো হয়।

---

# সবচেয়ে important ব্যাপার

Session server RAM-এ store হয় না।

বরং:

```text
State stored inside token
```

এজন্য এটাকে বলে:

```text
Stateless Authentication
```

---

# এখন distributed server এ কী হয়

```text
Request 1 → Server 1
Request 2 → Server 5
Request 3 → Server 2
```

সব server independently token verify করতে পারে।

কারণ:

- secret key same
    
- token self-contained
    

তাই:

- shared session দরকার নাই
    
- RAM synchronization দরকার নাই
