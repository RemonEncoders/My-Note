
# তাহলে logout problem কোথায়?

এখানেই আসে paradox।

কারণ JWT stateless।

মানে server session রাখছে না।

তাহলে:

```text
Logout করলে token invalid করবে কীভাবে?
```

Server তো token store-ই করে নাই।

---

# এটাই “Logout Paradox”

JWT naturally:

- login easy
    
- scaling easy
    

কিন্তু:

- immediate logout hard
    
- token revoke hard
